CSCB20 – Winter 2025

University of Toronto Scarborough

Introduction to Databases and Web Applications

Course Description:

The purpose of this course is to give students a practical introduction to databases and Web app development.

Topics:

  • Databases:
    • Key terminology and real-world applications
    • Creating, querying, and updating databases
    • Designing databases using the entity-relationship model
  • Web documents and applications:
    • Developing static and interactive web documents
    • Setting up web servers and generating dynamic server-side content
    • Integrating web applications with databases

Required Background

Experience with programming in an imperative language such as Python, Java, or C. This course may not be taken after or concurrently with any C- or D-level CSC course.

Motivation

Databases are integral to modern technology, supporting websites, applications, and online services. This course teaches students how to integrate databases with web applications.

Course Learning Outcomes

  • Understand the basic concepts and applications of database systems.
  • Learn relational database theory and relational algebra.
  • Master SQL for querying databases.
  • Develop database-driven web applications using HTML, JavaScript, Flask, and open-source tools.

Labs/Tutorials

Weekly in-person tutorials will be conducted.

Tentative Schedule

Week Topics

1-2 Relational Model, Relational Algebra
